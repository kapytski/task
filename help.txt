Исходный код для компиляции находится в файле MainFrame.java
После получения класса MainFrame он готов к работе.
При запуске на выполнение адрес txt-файла (например sample.txt) для чтения может быть передан в командной строке следующим образом:
"java MainFrame sample.txt". Если файл не обнаруживается в директории, откуда была запущена программа, предлагается вручную ввести адрес файла. По умолчанию словари для фильтрации текста ("english vcblr.txt","russian vcblr") находятся в той же директории, откуда запускается программа. Если словарь не обнаруживается, то предлагается ввести его адрес вручную. При ручном вводе адресов внешние кавычки и апострофы в именах файлов игнорируются.
